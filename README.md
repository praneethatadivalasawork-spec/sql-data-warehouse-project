# sql-data-warehouse-project
Building a modern data warehouse with sql , including ETL pre processes, data modeling, and analytics

## Project Requirements:

### Build the data warehouse

#### Objective:

develop the modern data warehouse using sql server to consolidate sales data , enabling analytical reporting and informed decision making.

#### Specifics:

- Data sources: import data from two source systems provided as csv files.
- Data Quality: cleanse and resolve data quality issues prior to analysis
- Integration: Combine both the sources and make a user friendly data model for analytical query
- Scope: Focus on the latests data, no historical require.
- Documentation.
